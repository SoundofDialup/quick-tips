Hide the File Path in Powershell

When you want to truncate the long file path that Powershell displays in VSCode/VSCodium.

Open/Create PowerShell Profile

[Option 1] Using Windows Explorer (create the file if necessary):

%userprofile%\Documents\WindowsPowerShell\Microsoft.PowerShell_profile.ps1

[Option 2] Using PowerShell:

PS C:\> Notepad $profile

Add Function to the Profile

function prompt {
    $p = Split-Path -leaf -path (Get-Location)
    "$p> "
}

Save and restart PowerShell.

Error: Not Allowed to Run Scripts

PS C:\> Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

Save and restart PowerShell.
