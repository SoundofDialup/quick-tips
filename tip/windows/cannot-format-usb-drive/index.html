Fix a USB Drive That Can't be Formatted

When trying to format, you get an error like "unknown capacity," or the drive is "write protected".

Run cmd as admin 

C:\> diskpart

DISKPART> list disk

Note the disk number for the USB drive, e.g. 1 

DISKPART> select disk 1

If the USB drive is write protected: 

DISKPART> attributes disk clear readonly

Remove the partition:

DISKPART> clean

Create a partition:

DISKPART> create partition primary

If the above gives an error: 

DISKPART> select partition 1

DISKPART> active

If it still doesn't work (this takes time): 

DISKPART> clean all

DISKPART> create partition primary

Format now (or do via Windows Explorer): 

DISKPART> format fs=fat32

Exit the DISKPART tool when finished and close the cmd window: 

DISKPART> exit
