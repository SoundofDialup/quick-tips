Connect to Home Server Running Fedora Server

On the home server

(Ensure that Fedora Server has been installed, obviously.)

Check if SSH is installed:

$ sudo dnf list installed | grep openssh-server

If not: 

$ sudo dnf install openssh-server

Start the SSH service:

$ sudo systemctl start sshd

Set things to auto-start SSH when server boots:

$ sudo systemctl enable sshd

Change port from default 22 (unless running with SELinux): 

$ sudo nano /etc/ssh/sshd_config

Uncomment # Port 22 and change to whatever.

Save config and restart service: 

$ sudo systemctl restart sshd

On the workstation

Access the server through terminal:

$ ssh [username]@[server IP address]

To connect to a non-default port: 

$ ssh [username]@[server ip address] -p [port]

ERROR: Cannot connect when using 127.0.0.x

Try using 192.168.x.x instead.
