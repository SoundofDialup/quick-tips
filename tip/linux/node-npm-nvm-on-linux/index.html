Install Node.js, npm, and nvm on Linux

Install Node.js

Use the Package Manager to install Node.js (see nvm below for version control): 

$ sudo apt install nodejs

Confirm installation:

$ node -v

Install npm

Use the Package Manager to install npm: 

$ sudo apt install npm

Install nvm

Manually run the install script to install nvm, which allows you to install different versions of node via the command line: 

$ curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash

Install node v 20 (restart IDE afterwards): 

$ nvm install 20

Optional: Use a specific version of node: 

$ nvm use 18
