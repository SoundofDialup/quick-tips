Install and Set up Tailwind CSS

Install Tailwind CSS

> npm install -D tailwindcss

(Ensure the appropriate version of Node.js is installed before next step.)

> npx tailwindcss init

Configure template paths

tailwind.config.js 
/** @type {import('tailwindcss').Config} */
module.exports = {
    content: ["./src/**/*.{html,js}"],
    theme: {
        extend: {},
    },
    plugins: [],
}

Create and Modify input.css

styles/input.css 
@tailwind base;
@tailwind components;
@tailwind utilities;

Start the CLI build process

> npx tailwindcss -i ./styles/input.css -o ./styles/output.css --watch

Add output.css to pages

/index.html 
<link href="./styles/output.css" rel="stylesheet">
