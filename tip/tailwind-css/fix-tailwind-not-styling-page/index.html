Fix for Tailwind CSS not Styling a Deployed Page

Check the deployed css file

It probably has the following:

styles.css 
@tailwind base;
@tailwind components;
@tailwind utilities;

Manually generate the css file 

> npx tailwindcss -i ./styles/input.css -o ./styles/output.css --minify

And make sure to link to output.css instead of styles.css .
