Add a Custom Domain to GitHub Pages

With your DNS Provider

Create a CNAME record pointing www to USERNAME.github.io

Create four A records pointing @ to:

185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153

With GitHub

Navigate to your repo and click the Settings tab.

Click Pages on the left.

Scroll down to Custom domain, type in your domain/url (e.g. example.com), and click Save.

Once the checks have been made, you can enforce HTTPS.
